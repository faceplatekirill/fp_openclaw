# ELECTRICITY-GRID Knowledge

This folder contains **electricity grid domain knowledge**.

## Scope
Use this folder for:
- Grid topology and connectivity interpretation
- Grid-specific semantics (`state_*`, QDS usage in grid operations)
- Grid workflows and operational patterns
- Grid graph model artifacts

## Scope Rule
If a topic depends on power-grid structure, equipment semantics, or dispatcher workflow,
it belongs in `ELECTRICITY-GRID/`.
